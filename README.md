# 🍔 Food Delivery Analysis using SQL

## 📌 Project Overview

This project analyzes food delivery data using **SQL** to uncover valuable business insights. The analysis focuses on customer ordering behavior, restaurant performance, delivery trends, and revenue generation. By writing SQL queries, the project answers real-world business questions and demonstrates practical SQL skills.

---

## 🎯 Objectives

* Analyze Impact of weather condition on delivery time .
* Analyze Impact of traffic on delivery time.
* Evaluate delivery performance.
* Track revenue and order trends.
* Generate business insights using SQL.

---

## 📂 Dataset

The dataset includes:

* Delivery Person Details
* Restaurant Information
* Orders
* Food Items
* Delivery Details
* Ratings
* Weather Condition

---

## 🛠️ Tools & Technologies

* SQL
* Power BI
* Microsoft Excel (for dataset)

---

## 📈 SQL Analysis

The project includes queries for:

* Total Orders
* Most Ordered Food Items
* Customer Spending Analysis
* Peak Ordering Hours
* Delivery Time Analysis
* Delivery Persons  Ratings
* Customer Order Frequency

---

## 💡 SQL Concepts Used

* SELECT Statements
* WHERE & ORDER BY
* GROUP BY & HAVING
* Aggregate Functions
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions

---

## 📊 Outcome

This project demonstrates how SQL can be used to analyze business data, extract meaningful insights, and support data-driven decision-making in the food delivery industry.
